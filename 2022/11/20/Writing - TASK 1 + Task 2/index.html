Writing - TASK 1

数据特征

动态特征：趋势，数据随时间推移的变化（上升、下降、波动、持平）

静态特征：差异，数据之间的高低差异（高低 高中低）

数据的具体选择方式

动态：起点，终点，极值点，拐点，交叉点

静态：最值，倍数，分数，居中，相似，差异值

小作文的分段

Section 1

一句话概括图形的大致内容，一般现在时，改写题目即可，一般放在文章开篇的第一句。

Section 2

1 ~ 2两个句子针对图形中最明显的动态和静态特征（明显特征可能不唯一）进行概括总结。只需要特征，并不需要具体数据值。

位置很灵活，可以选择放在开头段紧跟着内容介绍，也可以放在第二段独立成段，还可以放在文章结尾。

时态选择需要参照图形中数据出现的时间。

Section 3

根据数据特征将数据归类为2-3个类别，每个类别独立成段。

  1. 可以参考动态趋势差异 e.g. 上升 + 下降
  2. 可以参考静态高低差异 e.g. 高 + 低
  3. 按照逻辑差异来分组 + 数据上也存在差异体现 e.g. male + female

段落内部选择 核心特征 + 具体数据值。

时态需要参照图形中数据出现的时间。

地图题

地图题描述的事一个地区的变化，变化分为两种：图形原有事物的改变，图形新添事物。

图形原有事物的改变

原有事物：the original / previous / former

尺寸变大/变小：The size of A has been enlarged/extended/halved/reduced by half.

数量增多或减少：The number of A has been increased/risen/grown/reduced/decreased/dropped/fallen/doubled/tripled/quadrupled to 500.

原事物没了：completely disappeared/were removed

原事物被改为：A becomes B

A is transformed/reconstructed/redeveloped/converted/changed/turned over to/into B.

A is replaced/substituted by B.

A gives way to B.

图形新添事物

A newly built road

A new IT centre has been added to the library.

大作文段落结构

开头段 30-50词 背景 + 观点

纵向展开段 100词 提出一个核心理由 + 深入分析（证明观点合理）

横向展开段 100词 提出一个观点 + 2-3个理由罗列解释

驳论展开段 100词 提出反方观点 + 完成相应的驳论（观点不合理）

结尾段 20-30词 收尾

单边框架 = 开头 + 纵向 + 纵向 + 结尾

双边框架 = 开头 + 纵向/横向 + 纵向/横向 + 结尾

驳论框架 = 开头 + 纵向/横向 + 驳论 + 结尾

驳论段

100词左右对反方观点进行攻击或弱化

段落逻辑：

  1. 提出反方观点
  2. 提出反方观点成立的理由

  4. 结论 - 反方观点不太靠谱
